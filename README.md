# Sistema de Cadastro de Alunos

## Descrição

Projeto desenvolvido em linguagem C para gerenciamento de alunos utilizando CRUD.

O sistema permite:

- Cadastrar alunos
- Listar alunos
- Buscar alunos por ID
- Atualizar alunos
- Remover alunos

## Tecnologias

- Linguagem C
- GCC

## Estrutura

- Struct personalizada
- Vetor de 10 posições
- Controle da quantidade de registros
- Menu interativo

## Funcionalidades

### Cadastro
Insere um novo aluno validando ID único.

### Listagem
Exibe todos os alunos cadastrados.

### Busca
Localiza um aluno pelo ID.

### Atualização
Permite alterar nome e nota.

### Remoção
Remove um aluno mantendo a consistência do vetor.
